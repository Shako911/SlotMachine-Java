# Java Console Slots

A simple, interactive, and console-based Slot Machine game built with **Java**. The game tracks the player's balance, allows custom betting, validates inputs to prevent cheating or errors, and features different payout multipliers based on matching symbols.

---

## Features

- **Dynamic Payouts:** Multipliers change based on the rarity of the symbols (🍒, 🍉, 🍋, 🔔, ⭐).
- **Partial Wins:** Earn rewards for matching either 2 or 3 symbols in a row!
- **Robust Input Validation:** Prevents illegal bets (such as negative amounts, 0, or bets exceeding the current balance) without crashing the game.
- **Play Loop:** Play as long as you have money and choose to continue.

---

## Symbols & Payouts

| 🍋 Lemon | Bet × 4 | Bet × 5 |
